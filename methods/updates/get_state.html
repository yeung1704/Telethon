Bots can use this method. See code examples.

Known RPC errors

This request can cause 3 known errors:

AuthKeyDuplicatedErrorAn auth key with the same ID was already generated.
SessionPasswordNeededErrorTwo-steps verification is enabled and a password is required.
TimeoutErrorNo description known.

You can import these from telethon.errors.
