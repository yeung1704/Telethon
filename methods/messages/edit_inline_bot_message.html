Known RPC errors

This request can cause 2 known errors:

MessageIdInvalidErrorThe specified message ID is invalid.
MessageNotModifiedErrorContent of the message was not modified.

You can import these from telethon.errors.

Example

from telethon.sync import TelegramClient
from telethon import functions, types

with TelegramClient(name, api_id, api_hash) as client:
    result = client(functions.messages.EditInlineBotMessageRequest(
        id=types.InputBotInlineMessageID(
            dc_id=42,
            id=-12398745604826,
            access_hash=-12398745604826
        ),
        no_webpage=True,
        stop_geo_live=True,
        message='Hello there!',
        media=types.InputMediaUploadedPhoto(
            file=types.InputFile(
                id=-12398745604826,
                parts=42,
                name='some string here',
                md5_checksum='some string here'
            ),
            stickers=[types.InputDocument(
                id=-12398745604826,
                access_hash=-12398745604826
            )],
            ttl_seconds=42
        ),
        geo_point=types.InputGeoPoint(
            lat=7.13,
            long=7.13
        )
    ))
    print(result)
